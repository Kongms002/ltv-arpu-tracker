# Demo video

`grosshacker-demo.mp4` is a captioned, mock-data product walkthrough for a hackathon or Devpost submission.

Flow:

1. Growth brief and headline metrics
2. Cohort-level unit economics
3. Ingest → analyze → guardrail → activate pipeline
4. Explainable campaign recommendation review
5. Simulated budget action
6. Action audit trail
7. PostHog/GA4 and ad-network connector boundary

The video intentionally labels the current execution as a simulation. It does not claim that production ad accounts are connected.

To rerender after replacing frames:

```bash
bash demo/render_demo.sh
```
